# Gleam Jar

> *落ち穂拾いのように宝物をひろいあげる。あしたをちょっとだけ前向きにする願いをこめて・・・*

## コンセプト

日々の小さな「良かったこと」を書きとめて、瓶に貯めていくことができます。<br>
月ごとに書きとめた数に応じた天然石がもらえて、いつでも見かえすことができる自分だけの宝箱。<br>
<br>
Gleam Jar は「書いた事実」を守ります。<br>
落ち込んだときほど、過去の良かった記憶を否定したくなる。<br>
「あんなの良かったことじゃなかった」「浮かれてただけ」——<br>
そんなふうに思ってしまうタイミングで、せっかく貯めたものを衝動的に消してしまわないように。<br>
削除ではなく編集のみ。書き直しはできても、瓶の中身は消えません。<br>

## 主な機能

### (1) 書き込み
💜（ハート）: 内面の変化・自分についての良かったこと<br>
⭐（スター）: 外の出来事の良かったこと<br>
1日何個でも書ける（100文字程度 / 画像1枚まで）<br>

### (2) 石のシステム
月の書き込み数に応じて、もらえる石がグレードアップ<br>
少なめでも優しい石がもらえる<br>
何も書けなかった月は原石（ジオード）——それも立派な宝物<br>

### (3) ログ表示
日時で遡れる一覧<br>
💜 / ⭐ のフィルタリング<br>
キーワード検索<br>

### (4) マスコット
瓶のそばで見守る猫<br>
瓶が空のときも寂しくならないように<br>

### (5) ログイン・ユーザ登録
新規登録
ログインで自分のアカウントにアクセス

### (6) 通知機能
リマインダー
メール通知
