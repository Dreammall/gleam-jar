## Gleam Jar

あしたをちょっとだけ前向きにする、良かったこと貯金箱
日々の小さな「良かったこと」を書き留めて、瓶に貯めていくアプリです。
月ごとに天然石がもらえる、自分だけの宝物箱。

## コンセプト

落ち込んだときほど、過去の良かった記憶を否定したくなる。
「あんなの良かったことじゃなかった」「浮かれてただけ」——
そんなふうに思ってしまうタイミングで、せっかく貯めたものを衝動的に消してしまわないように。
Gleam Jar は「書いた事実」を守ります。
削除ではなく編集のみ。書き直しはできても、瓶の中身は消えない。
それがこのアプリの一番大切にしていること。

## 主な機能

書き込み

♡（ハート）: 内面の変化・自分についての良かったこと
⭐（スター）: 外の出来事の良かったこと
1日何個でも書ける（100文字程度 / 画像1枚まで）

石のシステム

月の書き込み数に応じて、もらえる石がグレードアップ
少なめでも優しい石がもらえる
何も書けなかった月は原石（ジオード）——それも立派な宝物

ログ表示

日時で遡れる一覧
♡ / ⭐ のフィルタリング
キーワード検索

マスコット

瓶のそばで見守る猫
瓶が空のときも寂しくならないように
